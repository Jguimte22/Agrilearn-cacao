# Game Audio Files

This folder contains audio files for the GAP Scramble Game.

## Required Audio Files

Place the following MP3 files in this directory:

1. **background.mp3** - Background music (loops during gameplay)
2. **correct.mp3** - Sound when answer is correct
3. **wrong.mp3** - Sound when answer is wrong
4. **powerup.mp3** - Sound when using power-ups
5. **levelup.mp3** - Sound when starting the game
6. **click.mp3** - Sound for button clicks
7. **gameover.mp3** - Sound when game ends

## Where to Find Free Audio Files

### Recommended Free Sources:
- **Pixabay**: https://pixabay.com/sound-effects/
- **Freesound**: https://freesound.org/
- **Mixkit**: https://mixkit.co/free-sound-effects/
- **Zapsplat**: https://www.zapsplat.com/

## Note
The game will work without audio files - it will just run silently if files are not found.
You'll see console warnings indicating which audio files couldn't be loaded.

## File Requirements
- Format: MP3
- File size: Keep under 1MB each for faster loading
- Sample rate: 44.1kHz or 48kHz recommended
